Create optimized childrens blocks

Creates 26 childrens blocks in the selected font optimized to create as many english language words as possible. Guaranteed to be able to form the whole alphabet and the numbers 0-9.

Click the get best faces button to create an optimized face set. This will draw out all the blocks. You can use the block size, padding margin and colums to modify how this is layed out. You can also select any of the google fonts you shich to use.

After you have all your settings set click the download button to get an svg for your laser cutter of choice.
